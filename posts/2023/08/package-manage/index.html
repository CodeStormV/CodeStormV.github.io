Python | 包管理

 2023.8.28  2023.10.20

安装所有 pip 包

pip install -r requirement.txt
-r 指定文件

删除所有 pip 的包

pip freeze > requirements.txt
freeze 是指将当前环境中安装的所有包及其版本信息列出，并将其输出到一个文本文件中 pip uninstall -r requirements.txt -y
-r 指定文件
-y 默认选择同意

pipreqs 导出所有依赖包

  1. 安装 pipreqs pip install pipreqs
  2. pipreqs ./ --encoding=utf-8 （导出所有项目依赖包，--encoding=utf-8 代表用 utf-8编码 来处理）
