VSCode | 文件转移

 2023.8.28  2023.10.20

文件转移

VSCode 会在 C 盘占用一部分空间，进行文件链接转移文件位置。 文件位置一般位于 C:\Users\用户名\AppData\Roaming\Code 和 C:\User\用户名\.vscode

操作：

  1. 剪切位于 C 盘的 VSCode 文件
  2. 粘贴到想要转移的位置
  3. 打开 cmd 输入
mklink /J "%APPDATA%\Code" "目标目录"
mklink /J "%USERPROFILE%\.vscode" "目标目录"

%APPDATA% 代表 C:\Users\用户名\AppData\Roaming
%USERPROFILE% 代表 C:\User\用户名

  1. 完成文件转移
